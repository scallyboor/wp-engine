How to install

1. Download Wallpaper Engine

2. Go to directory of Wallpaper Engine

3. Create a wallpaper64.exe shortcut

4. Enjoy!

How to download wallpapers

1. Download wallpaper

2. Open directory of Wallpaper Engine

3. Go to projects/myprojects

4. Create a new folder

5. Extract your wallpaper into the folder

6. Enjoy!
